# Regole di Progetto (Vibe Rules)

* **Styling**: Usa sempre Tailwind CSS.
* **Icone**: Preferisci icone Lucide-React.
* **Codice**: Scrivi codice pulito e commentato in italiano.
* **Framework**: Questo progetto usa Next.js e Payload CMS. Mantieni la compatibilità con le loro API.

> *Nota: Queste regole vengono lette automaticamente da Claude Code ogni volta che viene avviato in questa cartella.*
